# Machine Learning Engineer Capstone Project

The project utilises `scikit-learn` and `keras` libraries in Python

## Scripts
Two scrips are in the project folder,
*data_exploration* holds some visualisation
*data_script* includes all the code for building the database and running the
models.

## CSV files
All data is inside *csv_files* directory. These are required.

## Notebook

The jupyter notebook Predicting Premier League results from historic data.ipynb
has all the code correctly formatted.
Please use this file to view results or the html file. 

A pdf version is included but it was not formatted optimally.
